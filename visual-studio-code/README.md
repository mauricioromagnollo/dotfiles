# Visual Studio Code

## **Instalando a Fonte (Fira Code)**

### Debian / Ubuntu

```bash
sudo apt install fonts-firacode -y
```

### Arch / Manjaro

```bash
sudo pacman -S ttf-fira-code --noconfirm
```

### RedHat / Fedora / CentOS

```bash
sudo dnf copr enable evana/fira-code-fonts

sudo dnf install fira-code-fonts -y
```
